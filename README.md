# MiSTer IFS Fractals

### Intro
An Iterated Function Systems (IFS) renderer core for the MiSTer FPGA platform written in Verilog.

### Installation
To install it, copy the .rbf file from the **[Releases](https://github.com/movievertigo/MiSTer-IFSFractals/releases)** page to your MiSTer.

### Controls
The core is interactive, and you can map the following controls:
* **Left,Right,Up,Down :** *Pan controls*
* **In,Out :** *Zoom controls*
* **Toggle Clear :** *Toggles screen clearing*
* **Toggle Random :** *Toggles resetting of the random number generator each frame*
* **Toggle Shades :** *Toggles between pure black and white and greyscale*
* **Next Preset,Previous Preset :** *Change the fractal type*
